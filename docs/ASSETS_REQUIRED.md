# 所需素材清单

> 由于版权原因，本仓库不包含实际素材文件。请自行准备以下文件，放置于 `assets/` 目录下。

## 素材列表

| 文件名 | 类型 | 推荐规格/说明 |
|--------|------|----------------|
| bg_janus.png | 图片 | 1280×720 像素 |
| door_sprite.png | 图片 | 582×540 像素 |
| ev_reader.png | 图片 | 1280×720 像素 |
| ev_writer.png | 图片 | 1280×720 像素 |
| simkai.ttf | 字体 | TrueType/OpenType 字体文件 |

## 合法素材获取途径

- **字体**：推荐使用 SIL OFL 协议的开源字体，如 [思源黑体](https://github.com/adobe-fonts/source-han-sans)、[霞鹜文楷](https://github.com/lxgw/LxgwWenKai)
- **图片**：可从 [Pixabay](https://pixabay.com)、[OpenGameArt](https://opengameart.org) 查找 CC0 或 CC-BY 授权的图片
- **音频**：[Freesound](https://freesound.org) 提供大量 CC 授权音效

## 使用说明

1. 创建 `assets/` 文件夹（如不存在）
2. 将上述列表中的文件放入该文件夹
3. 重新运行项目即可
